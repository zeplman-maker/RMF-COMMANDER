2. Control Principles
Human-in-the-loop for every AI-generated RMF artifact.
Least privilege access based on authenticated role scope.
No client-side persistence of authorization package content.
Full auditability for API calls, AI prompts, generated outputs, and approvals.
Separate pilot approval from operational ATO approval. These are distinct gates.
