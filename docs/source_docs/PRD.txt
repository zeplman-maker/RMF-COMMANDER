1. Product Vision
RMF COMMANDER is an AI-assisted mobile and web platform that gives authorized RMF personnel secure portfolio visibility, smart compliance alerts, AI-supported documentation drafting, natural-language RMF query, and executive briefing support. The system extends visibility and drafting support without replacing eMASS or authorization judgment.

4. Non-Functional Requirements
Area
Requirement
Data classification
IL4 handling in a DoW-authorized cloud boundary. No classified data in pilot.
Authentication
CAC/PIV-derived credential via Purebred on MDM-enrolled devices.
Session management
15-minute JWT and full wipe after 5 minutes backgrounded or logout.
Data at rest
No eMASS data persisted on the client device. Session memory only.
Transport
TLS 1.3 minimum and mTLS for backend-to-eMASS.
AI boundary
No IL4 data transmitted outside the DoW cloud boundary for AI processing.
Audit logging
All API calls, AI prompts, outputs, and approvals logged with 1-year minimum retention.
STIG compliance
DISA Mobile App STIG review before operational deployment.

6. Acceptance Criteria
A pilot user can view full mock portfolio posture in less than five minutes.
Configured 90/60/30-day ATO alerts fire with 100 percent accuracy in test.
AI-generated drafts are labeled AI-drafted, source-cited, and blocked from use until human approval.
No production eMASS data is used during pilot.
No client device persists authorization package content.
AO receives 30-day touchpoint briefings and final go/no-go report.
